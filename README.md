Sobre

Projeto desenvolvido como parte do curso da FATEC.
Este repositório contém um sistema web que utiliza HTML, CSS e JavaScript.

Estrutura do repositório
/CSS/        → arquivos de estilo (.css)  
/imgs/       → imagens utilizadas no site  
index.html   → página inicial  
login.html   → página de login  
cadastro.html → página de cadastro  
doacaoUsuario.html → página de doação para usuário  
doacaoEmpresa.html → página de doação para empresa  
mapa.html    → página de mapa / localização  
sobre.html   → página “Sobre” o projeto  
CalculoAgua.html → página de cálculo de água (ou outra funcionalidade)  
form.js / script.js → scripts JavaScript para funcionalidades do site  

Tecnologias utilizadas

HTML

CSS

JavaScript

Como usar / executar localmente

Clone o repositório:

git clone https://github.com/Enzo-Belizardo/PI_Fatec.git


Abra o arquivo index.html no navegador utilizando o aplicativo "Visual Studio Code" Com a extensão "Live Server"

Para editar e desenvolver, faça modificações nos arquivos correspondentes em CSS/, imgs/, e arquivos .html ou .js.
